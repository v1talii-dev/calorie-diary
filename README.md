# calorie-diary

Дневник калорий:

- Для мобильных устройств

## Установка

> Актуально для версии node.js `v20.18.0`, npm `v10.8.2`.

```sh
npm install
```

## Разработка

**Создайте файл `.env` на основе `env.example` и заполните соответствующие переменные**

```sh
npm run dev
```

### Сборка

```sh
npm run build
```
